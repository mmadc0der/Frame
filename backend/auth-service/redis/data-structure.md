# Note for redis data organisation

```sql
+------------------------------------------------------+
|                       users                          |
+------------------------------------------------------+
| id (PK)                                             |  <-- Уникальный идентификатор пользователя
| username                                            |  <-- Логин пользователя
| email                                               |  <-- Почта пользователя
| email_verified (BOOLEAN)                            |  <-- Флаг подтверждения email
| password_hash (NULLABLE, если OAuth, то NULL)       |  <-- Пароль
| created_at                                          |  <-- Дата регистрации
| last_login_at                                       |  <-- Последний вход
| full_name                                           |  <-- ФИО (если предоставляется)
| phone_number (NULLABLE)                             |  <-- Номер телефона
| gender (NULLABLE)                                   |  <-- Пол
| birth_date (NULLABLE)                               |  <-- Дата рождения
| provider (VARCHAR, NULLABLE)                        |  <-- OAuth-провайдер
| provider_user_id (VARCHAR, NULLABLE)                |  <-- ID пользователя в OAuth-сервисе
| provider_data (JSONB, NULLABLE)                     |  <-- Дополнительные данные от провайдера
+------------------------------------------------------+

+------------------------------------------------------+
|                       roles                          |
+------------------------------------------------------+
| id (PK)                                             |  <-- Уникальный ID роли
| name (VARCHAR, UNIQUE)                              |  <-- Имя роли (например, "admin", "user")
| description                                         |  <-- Описание роли
+------------------------------------------------------+

+------------------------------------------------------+
|                    permissions                       |
+------------------------------------------------------+
| id (PK)                                             |  <-- Уникальный ID права
| name (VARCHAR, UNIQUE)                              |  <-- Имя права (например, "view_logs")
| description                                         |  <-- Описание права
+------------------------------------------------------+

+------------------------------------------------------+
|                role_permissions                      |
+------------------------------------------------------+
| id (PK)                                             |  <-- Уникальный ID
| role_id (FK to roles.id)                            |  <-- ID роли
| permission_id (FK to permissions.id)                |  <-- ID права
+------------------------------------------------------+

+------------------------------------------------------+
|                    user_roles                        |
+------------------------------------------------------+
| id (PK)                                             |  <-- Уникальный ID
| user_id (FK to users.id)                            |  <-- ID пользователя
| role_id (FK to roles.id)                            |  <-- ID роли
+------------------------------------------------------+

```
